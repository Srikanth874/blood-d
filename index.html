Frequently Asked Questions

Short answers for common questions about donation and requests.

How does this website store donor data?

It uses Netlify Forms for simple form submission storage after deployment. That is enough for a working college project.

Can I add real database later?

Yes. You can connect Firebase, Supabase, or a Node.js backend with Neon/PostgreSQL later if your project needs a database.

Will it work on mobile?

Yes. The layout is fully responsive and automatically adjusts for phone, tablet, and desktop screens.
